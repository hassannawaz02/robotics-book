# Diagrams for Module 2: Digital Twin (Gazebo & Unity)

This directory contains architecture diagrams and visual representations used in the Digital Twin module lessons.

## Diagram Types

- System architecture diagrams
- Workflow diagrams
- Component interaction diagrams
- Data flow diagrams
- Environment setup diagrams

## Diagram Guidelines

When creating diagrams for this module:

- Use consistent styling and color schemes
- Include clear labels and legends
- Provide both high-level overviews and detailed views
- Ensure diagrams are accessible with proper alt text
