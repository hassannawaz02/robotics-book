## Overview of the Isaac Ecosystem

The NVIDIA Isaac ecosystem provides a comprehensive set of tools for developing, testing, and deploying robotics applications:

![Isaac Architecture](../assets/diagrams/isaac-architecture.svg)

The ecosystem consists of three main components that work together to enable sophisticated robotics applications:

1. **Isaac Sim** - A photorealistic simulation environment for testing and training
2. **Isaac ROS** - Hardware-accelerated perception and navigation algorithms
3. **Nav2** - Advanced path planning capabilities, especially for humanoid robots

In the following lessons, we'll explore each component in detail and learn how to integrate them into a complete AI-robot brain system.
