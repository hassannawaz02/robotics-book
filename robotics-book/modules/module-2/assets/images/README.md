# Images for Module 2: Digital Twin (Gazebo & Unity)

This directory contains images used in the Digital Twin module lessons.

## Available Images

- Placeholder images for:
  - Gazebo environment screenshots
  - Unity scene captures
  - Sensor data visualizations
  - Architecture diagrams
  - Workflow illustrations

## Image Guidelines

When adding images to this module:

- Use descriptive filenames
- Maintain consistent resolution (recommended 1920x1080 for full-width images)
- Optimize file sizes for web delivery
- Include alternative text descriptions
